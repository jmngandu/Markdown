When creating a list you can use two formatting styles, using "-" or "*"
for UNORDERED LISTS:
  using "-"
        - Africa
        - Asia
        - America

  using "*"
        *Africa
        *Asia
        *America

   using html:
         <ul>
            <li>Africa</li>
            <li>Asia</li>
            <li>America</li>
        </ul>
for ORDERED LISTS:
    
        1. Africa
        2. Asia
        3. America
    
   using html:
        <ol>
            <li>Africa</li>
            <li>Asia</li>
            <li>America</li>
        </ol>

for CHECKLIST:
we use [] to show unmarked checklist but for marked checklist we show by use of [x]    
 for example
- [x] Africa
- [ ] America
- [ ] Asia

for NESTED LISTS:
        - Africa
           - East africa
           - West africa
           - Central africa
        - Asia
        - America
